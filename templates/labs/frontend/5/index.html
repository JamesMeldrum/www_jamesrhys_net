  • Web and Desktop code is parallel with user-agent flags in Apache
  • Device-specific languages adhere to interfaces not frameworks
  • Processing farmed out to devices
  • More tools available to debug JS than *SOME* popular web languages
  • JavaScript done correctly is beautiful
